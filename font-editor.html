Encoding

6 bytes = 6 columns (col 5 = spacing).
Byte = column bitmask.
Bit 0 (LSB) = row 0 (top).
Bit 6 = row 6 (bottom).
Bit 7 always 0 (dimmed row).
Row 7 = unused (shown dimmed).
